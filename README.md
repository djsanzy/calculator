# calculator

This calculator is 100% python with an ascii art. It just has the add, subtract, multiply
and divide finction which has a return output
I created the operations using function and used it in a dictionary.
Looped the dictionary to get the key from the users choice and assigned it to a variable. But to get the user choice in order to perform an action from it, i had to get the key 
matching the users choice.
Had to use a while loop to get the operations repeat and also be able to get the user type 
either yes or no. 
If the user types yes, and chooses an operation, the calculator uses the answer it get to keep calculating but if the user types no, the calculator starts afresh.
This is done using recursion. Recursion is wrapping all or some code that you want repeating when it is triggered
